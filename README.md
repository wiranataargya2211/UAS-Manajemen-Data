# Tugas Ujian Akhir Manajemen Data
Repositori ini berisi dokumentasi dan source code untuk penyelesaian tugas mata kuliah Manajemen Data.

## Daftar Isi
1. **Tugas 1 (Linux System Administration):** Berisi *bash script* untuk pembuatan *user* otomatis, pengaturan ruang *backup* (mounting), penjadwalan crontab, dan pengecekan kapasitas *hardisk*.
2. **Tugas 2 (Docker Container):** Berisi implementasi *containerization* menggunakan Docker untuk membungkus aplikasi web Flask (Python) bernama **Student Score Analyzer** yang berfungsi untuk mengkalkulasi dan memeringkatkan data nilai mahasiswa.
